# VHF

a random .wav from tape is chosen w/ random granular parameters on every channel. VHF uses softcut.

## Controls

k1 (hold): tv guide  
k2: select sample file. also sets file directory for subsequent channel changes  
k3: change channel  
e1: volume  
e2: change sample speed + randomize softcut 1
e3: change sample pitch + randomize softcut 2
